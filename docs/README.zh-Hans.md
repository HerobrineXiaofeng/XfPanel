## 快速开始
> **要求:** Linux VPS (Debian / Ubuntu / CentOS / Rocky)、1 GB 内存、互联网接入。  
> 需要 ~3 分钟。
```bash
bash <(curl -fsSL https://dl.xfpanel.com/xfpanel/package/v1/install.sh)
```
