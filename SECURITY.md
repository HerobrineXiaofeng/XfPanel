# Security Policy

## Supported Versions
We provide security updates for the following project versions.
Please upgrade to a supported version to receive security fixes.

| Version | Security Support |
| ------- | ---------------- |
| Latest  | ✅ Supported      |
| Legacy  | ❌ Not Supported |

## Reporting a Vulnerability
Please **do not report security vulnerabilities through public Issues, Discussions or Comments**.

### Report Channel
1. Preferred: Private security report channel of the platform repository
2. Email: support@xfpanel.com

Please include detailed information:
- Vulnerability description and impact scope
- Reproduction steps or PoC (if available)
- Affected version and code location
- Your contact way for follow-up communication

## Response & Disclosure
- We will confirm receipt of vulnerability reports within 48 hours.
- We will give assessment and follow-up progress regularly.
- After the vulnerability is fixed, we will publish related updates and acknowledgments.
- Please do not disclose vulnerability details publicly before official repair and announcement.

## Security Update
Security patches will be released in routine version updates.
Critical security issues will be fixed as priority.
