## 一键安装 (正式版)
```bash
bash <(curl -fsSL https://dl.xfpanel.com/xfpanel/package/v1/install.sh)
```
## 一键安装 (dev版)
```bash
bash <(curl -fsSL https://dl.xfpanel.com/xfpanel/package/v1/install.sh) --dev
```
