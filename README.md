## Quick Start
> **Requirements:** Linux VPS (Debian / Ubuntu / CentOS / Rocky), 1 GB RAM, internet access.  
> Takes ~3 seconds.
```bash
bash <(curl -fsSL https://dl.xfpanel.com/xfpanel/package/v1/install.sh)
```
